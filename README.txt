1. Open the server in the respective server folder by typing python server.py <Server Port>
2. Open the clients in the resepective client folder by typing python server.py <Server Machine> <Server Port>
3. Then type ? in order to see the commands.
4. Note: error handling has not been done.
5. When finished, close the server by hitting control+C
6. Type quit into each of the client or clients running.
